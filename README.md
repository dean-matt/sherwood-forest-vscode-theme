# Sherwood Forest

A dark VS Code color theme that is based around forest colors.

Contains a theme for each season: Spring, Summer, Fall, and Winter.

## Summer

### Primary Palette:

```css
primary {
  --black: #000000;
  --night: #030f0e;
  --dark-green: #0c3632;
  --sea-green: #008f62;
  --emerald: #00cc8b;
  --white: #ffffff;
}
```

![Primary Palette](<themes/Summer Palette (Primary).png>)

### Secondary Palette:

```css
secondary {
  --beaver: #827564;
  --sepia: #522e00;
  --chocolate: #7a4500;
  --copper: #b08246;

}
```
